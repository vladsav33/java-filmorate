# java-filmorate
Cервис для работы с фильмами и оценками пользователей

# Реализованные фичи
* Отзывы
* Рекомендации
* Лента событий
* Общие фильмы
* Поиск
* Удаление фильмов и пользователей
* Вывод самых популярных фильмов по жанру и годам
* Добавление режиссеров в фильмы
